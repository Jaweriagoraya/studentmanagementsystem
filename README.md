# 🎓 Student Management System (OOP Java)

A **Java-based Student Management System** developed using Object-Oriented Programming (OOP) principles.  
This project is designed to manage student records efficiently with database integration.

---

## 🚀 Features

- Add new student records  
- Update existing student details  
- Delete student records  
- Search students by ID or name  
- Display all student data  
- Simple and user-friendly interface (Console or GUI based)  
- Database connectivity using JDBC  

---

## 🛠️ Technologies Used

- Java (OOP Concepts)  
- JDBC (Java Database Connectivity)  
- MySQL Database  
- IntelliJ IDEA (IDE)

---

## 🗄️ Database Information

This project uses a MySQL database to store student information.
